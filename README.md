# Project Grafika Komputer

Repository ini dibuat untuk memenuhi tugas mata kuliah Grafika Komputer.

# Deskripsi

Projek ini berisi dokumentasi pembuatan akun GitHub dan repository yang digunakan sebagai media pengumpulan tugas mata kuliah Grafika Komputer.

# Cara Menggunakan

1. Buka repository ini melalui link GitHub

2. Baca dokumentasi yang tersedia

3. Repository digunakan sebagai tempat penyimpanan dan pengumpulan tugas mata kuliah
